# Functional Requirement Specification (FRS)
## Project: E-Commerce Website (demo.opencart.com)
### Version: 1.0
### Author: Shivam Verma

## 1. Introduction
This FRS describes the functional requirements for the demo e-commerce website used for manual testing.

## 2. Features / Functional Requirements
1. **User Authentication**
   - FR-01: User can register with name, email, password.
   - FR-02: User can login using valid credentials.
   - FR-03: Password reset via email.

2. **Product Catalog**
   - FR-04: Products are listed with name, price, image.
   - FR-05: Search by product name returns relevant results.
   - FR-06: Filter by category/price sorts correctly.

3. **Shopping Cart**
   - FR-07: Add product to cart.
   - FR-08: Update product quantity in cart.
   - FR-09: Remove product from cart.

4. **Checkout & Orders**
   - FR-10: Checkout guest/registered user.
   - FR-11: Enter shipping and billing details.
   - FR-12: Order confirmation and order history.

5. **Payments**
   - FR-13: Payment completion via demo payment method or order placement simulation.
